========================================================================
    C++/WinRT rips (ripper server) Project Overview
========================================================================

This project demonstrates a C++/WinRT console app network wrapper 
around the ripper server object.

The C++/WinRT networking portion could easily be removed and the server
built into a custom C++ app, Windows Runtime Component, or linux app.

========================================================================
Learn more about C++/WinRT here:
http://aka.ms/cppwinrt/
========================================================================
